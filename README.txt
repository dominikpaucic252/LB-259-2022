# LB-259-2022

Der Datensatz ist eine Sammlung von etwas mehr als 20.000 Partien, die von einer Auswahl von Benutzern auf der Website Lichess.org gesammelt wurden.
Die Daten bestehen aus Spiel-ID; Bewertet (T/F); Startzeit; Endzeit; Anzahl der Züge; Spielstatus; Gewinner; Zeitinkrement; Weiße Spieler-ID; Weiße Spielerbewertung;
Schwarze Spieler-ID; Schwarze Spielerbewertung; Alle Züge in Standard-Schachnotation; Eröffnung Eco (Standardisierter Code für jede gegebene Öffnung,Liste hier );
Eröffnungsname; Opening Ply (Anzahl Züge in der Eröffnungsphase)
